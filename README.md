### Requirements

- [Node.js](https://nodejs.org) v.18.0.0^

### Install packages

```sh
npm install
```

### Build production

```sh
npm run build
```

### Run production server

```sh
npm run start
```

### Run development hot-reload server

```sh
npm run dev
```

### Development hints
